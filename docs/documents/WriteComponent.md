# 编写组件文档

1. 在`components`下添加对应的md
2. 添加完对应md后，再在对应的examples下添加对应的`md名称的文件夹`
3. md文件下，如果你需要组件实例，请按照以下格式

```ts
::: demo
在对应在examples下的组件
:::
```
4. 并在对应的examples写出组件
